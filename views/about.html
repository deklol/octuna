❓ About Octuna

Octuna is a tiny image host you can run on your own subdomain. It exists because every other "free" image host either started charging, started showing ads, or quietly deleted your files after a few months.

Octuna doesn't do any of that. There are no accounts, no ads, no tracking, no expiration. Upload a file, get a short URL, share it. That's the whole product.

The frontend is dressed up like a 1999 GeoCities-meets-Internet-Explorer site because it's funny and because pretending we have unlimited bandwidth and a 3D animated logo is more fun than another sterile drag-and-drop rectangle.

The backend is a single ~250-line Node script with one dependency. It uses about 20 MB of RAM. You can run it on a Raspberry Pi.
